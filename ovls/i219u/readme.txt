i219u.ovl

Драйвер для работы с датчиком INA219 по UDP

Инициализация:
mdb[60] - Номер пина (0..15) для сигнала SCL, по умолчанию GPIO4
mdb[61] - Номер пина (0..15) для сигнала SDA, по умолчанию GPIO5
Переменные:
mdb[62] - Счетчик опроса датчика. При смене указывает, что с датчика обработаны новые данные. 
mdb[63] - Флаг: 
			=0 - данные верны, ошибок нет. 
			=-1 - драйвер закрыт.
			=-2 - неисправность сенсора
			=-3 - ошибки в параметрах инициализации (номера пинов SCL, SDA)
mdb[64] - напряжение (1 мВ)
mdb[67] - сенсор тока (0.1 мА)

mdb[70] - dword ADC sample rate 1..2000 SPS
mdb[72] - dword host ip
mdb[74] - word host port
mdb[75] - word remote port
Переменные:
mdb[76] - Флаг:	
			=0 - драйвер закрыт
			=1 - драйвер установлен и работает 

WebSocket:  
ovl$2 – отключить сканирование для UDP
ovl$3 – получить значение напряжения в 0.5 мВ
ovl$4 – получить значение тока в 0.1 мА


UDP Запрос / Выходная информация или команда
'?'	WDRV: ver	
'F=fx'	WDRV: freg fx Hz	
'A'	WDRV: set remote ip , remote port	
'S'	WDRV: stop	
'G'	Start wave
